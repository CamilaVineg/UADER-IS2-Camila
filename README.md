# UADER-IS2-Camila

## Propósito del repositorio

Este repositorio se utilizará para desarrollar y almacenar proyectos y talleres solicitados en la materia **Ingeniería de Software 2** de la carrera **Licenciatura en Sistemas de Información** en la universidad **UADER FCyT**.

### Organización del repositorio

El repositorio está estructurado en carpetas y archivos que representan distintos ejercicios y prácticas:

1. **src/**  
   Contiene los programas principales (`factorial.py`, `factorial_OOP.py`, `Collatz.py`, `primes.py`).
2. **docs/**  
   Espacio reservado para documentación adicional, guías y apuntes.
3. **bin/**  
   Carpeta destinada a archivos ejecutables o scripts compilados.
4. **script/**  
   Carpeta para almacenar scripts auxiliares y automatizaciones.

### Proyectos incluidos

- **Factorial clásico**: cálculo de factoriales con entrada manual y por argumentos.
- **Factorial orientado a objetos**: implementación con clases y métodos.
- **Collatz**: cálculo de iteraciones de la conjetura de Collatz y generación de gráficos.
- **Primes**: programa para trabajar con números primos.
